# یادداشت فارسی پروژه

این ریپو برای تمرین و تست سریع مدل‌های classification ساخته شده است.

ارتباطش با مقاله IoT/Windows این است که قبل از اینکه مدل‌ها را وارد پروژه اصلی کنم، اینجا با دیتای کوچک تست می‌کنم تا مطمئن شوم کد، متریک‌ها، train/test split و ذخیره نتایج درست کار می‌کند.

## معنی Playground

در اینجا playground یعنی محیط تمرینی یا آزمایشگاهی. یعنی جایی برای تست کردن، خراب کردن، اصلاح کردن و مقایسه کردن مدل‌ها.

## نکته مهم

این ریپو قرار نیست جای مقاله اصلی باشد. فقط کمک می‌کند ایده‌ها را سریع‌تر و تمیزتر تست کنم.
