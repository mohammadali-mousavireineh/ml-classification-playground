# Data

This folder contains only small sample data so the project can run after cloning.

The file `sample_iot_windows.csv` is artificial and should not be used as a real benchmark. It only follows the shape of a tabular network/security classification task.

For the real paper-related experiments:

1. keep raw datasets outside GitHub if they are large or private
2. document the feature columns in `docs/paper-connection.md`
3. save only small processed samples or result tables when possible
